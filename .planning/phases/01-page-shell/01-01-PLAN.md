  <action>
Prism.js must be loaded from CDN with the correct language components for Go and CUE.

Add to <head> (before </head>):
```html
<link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/prism/1.29.0/themes/prism-tomorrow.min.css">
```

Add before </body>:
```html
<script src="https://cdnjs.cloudflare.com/ajax/libs/prism/1.29.0/prism.min.js"></script>
<script src="https://cdnjs.cloudflare.com/ajax/libs/prism/1.29.0/components/prism-go.min.js"></script>
```

CUE does not have a Prism.js language component. Use a custom minimal language registration to give CUE blocks distinct highlighting. Add this inline <script> after the Prism core script:

```javascript
Prism.languages.cue = {
  'comment': /\/\/.*/,
  'string': /"(?:[^"\\]|\\.)*"/,
  'keyword': /\b(?:if|for|let|import|package|true|false|null|_|close)\b/,
  'operator': /[=:&|!<>+\-*\/]+/,
  'punctuation': /[{}[\](),]/,
  'number': /\b\d+(?:\.\d+)?\b/,
  'builtin': /\b(?:string|int|bool|float|bytes|list|struct)\b/,
};
```

This ensures:
- Go blocks get full keyword, type, and function highlighting (orange/purple/blue from prism-tomorrow theme)
- CUE blocks get comment (grey), string (yellow), keyword (purple), operator (white), builtin (cyan) coloring — visually distinct from Go

Verify the two language classes are already applied to placeholder code blocks from Task 1. Do not change code content — only ensure the CDN links and language registration are correctly in place.

Also add a sidebar active-link highlight script. Inject before </body>:
```javascript
const sections = document.querySelectorAll('section[id]');
const navLinks = document.querySelectorAll('#sidebar a[href^="#"]');
window.addEventListener('scroll', () => {
  let current = '';
  sections.forEach(s => {
    if (window.scrollY >= s.offsetTop - 120) current = s.id;
  });
  navLinks.forEach(a => {
    a.classList.toggle('active', a.getAttribute('href') === '#' + current);
  });
});
```

Add CSS for the active class in the existing <style> block:
```css
#sidebar a.active {
  background: #313244;
  border-left: 3px solid #89b4fa;
  color: #89b4fa;
  padding-left: calc(1rem - 3px);
}
```
